# 1️⃣ What is the difference between var, let, and const?
Answer:var: is the old way to create variables in JavaScript.It can be re-declared and re-assigned.
let:let is the modern way to create variables.It cannot be re-declared, but it can be re-assigned.It has block scope.
const:const is used to create constants value.It cannot be re-declared or re-assigned.It also block scope Variable.

# 2️⃣ What is the spread operator (...)?
Answer:The spread operator (...) in JavaScript is used to expand or spread the elements of an array or object into another array or object.

# 3️⃣ What is the difference between map(), filter(), and forEach()?
Answer:map(): is used to create a new array by modifying each element of the original array.
filter(): is used to select elements that match a condition.
forEach(): is used to run a function for each element of an array.

# 4️⃣ What is an arrow function?
Answer:An arrow function is  shorter and simpler way to write a function in JavaScript. It was introduced in ES6 and uses the => (arrow) symbol to define a function.

# 5️⃣ What are template literals?
Answer: Template literals are a feature in JavaScript used to create strings easily and dynamically.
